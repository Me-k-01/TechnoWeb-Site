Qu'est ce qu'un package ?

Les packages sont des extensions développées par la communauté fournissant une grande variété de fonctionnalité à Atom. Développées en Node.js, ils sont facilement installables depuis la rubrique installation des paramètres, mais une fois installés, ils sont accessibles et configurables depuis leur rubrique dédiée: "packages" des paramètres.

Recommendation

Dans cette section, je vous conseillerais quelques packages que je trouve utiles.

  • Script

    Qui permet d'exécuter vos scripts confortablement depuis l'éditeur. Fonctionnant avec une très grande partie des langages de programmation.

    Attention! Il vous faut tout de même que la commande liée à l'exécutercution de votre script (ex: python script.py) soit accessible comme variable d'environnement grace au PATH. Car tous ce que fait cette extension est de lancer votre script dans un terminal.

  • BrowserPlus

    Pour pouvoir travailler facilement sur ces projets web, avec notamment un rafraîchisseur automatique de page que l'on peut activer a souhait (icon d'eclaire)

  • Linter

    Pour voir ses erreurs de syntaxe et autres s'afficher de manière dynamique dans l'éditeur.

  • Color Picker

    Pour choisir une couleur de manière simple et avoir accès aux différents codes de celles-ci (rgb, hexadecimal...)

  • Teletype

    Pour travailler à plusieurs sur un même fichier.

  • Ask Stack

    Pour rapidement avoir accès au forum de Stack Overflow depuis l'éditeur

  • AutoComplete-plus

    Qui permet d'ajouter manuellement des suggestions que l'éditeur pourra vous proposer par la suite lors de la saisie.

  • Pigments

    Pour voir les codes couleurs se surligner de leurs couleurs correspondantes.

  • Atom Live Server

    Pour générer un serveur http local.

  • Minimap

    Pour pouvoir naviguer sans s'y perdre lorsque l'on a en face de soi un script particulièrement long.
