Téléchargement

Atom est disponible en téléchargement pour Windows, macOS et Linux via le site officiel.

L'installation se resume à ouvrir l'exécutable téléchargée. Démonstration

Il vous sera proposé d'agréer à l'envoi des données télémétriques, que vous pouvez choisir d'accepter ou de refuser. Je vous conseille simplement de fermer tous ces onglets inutiles, pour plus de simplicité dans la suite du tutoriel.

Configuration

Les paramètres sont accessibles en appuyant simultanément sur les touches "Ctrl" et ",". Il est également accessible depuis le menu du haut File > Settings pour Windows ou depuis Edit > Preferences pour Linux.

Une fois arrivé sur cette fenêtre vous aurez des options qui s'afficheront devant vous. Je vais en souligner quelques une qui me semble intérêssant d'activer.

  • Auto Hide Menu Bar

    Simplement parce que cela peut être quelque peu distrayant d'avoir une bande blanche au-dessus de ce que l'on code (ne vous inquiétez pas, le menu sera toujours accessible en appuyant sur Alt)

  • Restore Previous Window On Start

    Pour un retour rapide au travail.

Pour le reste je vous laisse décider par vous-même de ce que vous voulez.
