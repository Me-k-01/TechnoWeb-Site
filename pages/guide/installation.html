Téléchargement

Tout d'abord vous pouvez télécharger Atom sur Windows, Mac os et Linux sur le site officiel.

L'installation se résume a appuyer sur suivant, suivant, suivant et accepter les thermes d'utilisation de l’éditeur.

Il vous sera proposez d'agreer a l'envoie des données telemetrique, Je vous conseil simplement de fermer toute ces pages inutiles

Configuration

Les parametres sont accessible en appuyant sur les touches Ctrl + Virgule. Il est égalemment accessible depuis le menu du haut File > settings pour Windows ou depuis Edit > preférences pour Linux.

Une fois arrivé sur cette fenêttre vous aurez des options qui s'afficherons devant vous

Je vais en souligner quelque une que je trouve digne d'interet:

  • Auto Hide Menu Bar

    Simplement parce que cela peut etre quelque peux distrayant d'avoir une bande blanche au dessus de ce que l'on code (ne vous inquietez pas il sera toujours accessible en appuyant sur Alt)

  • Restore Previous Window On Start

    Pour un retour rapide au travail.

Pour le reste je vous laisse dessider par vous meme de ce que vous voulez.
